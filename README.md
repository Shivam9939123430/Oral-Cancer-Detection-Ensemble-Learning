# Oral Cancer Detection Using Ensemble Deep Learning

A deep learning-based research project for early oral cancer detection using an ensemble of multiple CNN architectures.


##  Project Overview

This project compares the performance of multiple convolutional neural networks and combines them using Ensemble Learning to improve oral cancer classification accuracy.

The models used include:

- ResNet
- DenseNet
- EfficientNet
- Ensemble Learning

##  Features

- Medical image classification
- Transfer Learning
- Ensemble Deep Learning
- Performance Comparison
- Model Evaluation
- Confusion Matrix
- Accuracy & Loss Graphs
